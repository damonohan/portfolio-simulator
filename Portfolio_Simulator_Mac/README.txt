===== PORTFOLIO SIMULATOR =====

HOW TO RUN THE APPLICATION:

1. Double-click the "Install and Run Portfolio Simulator" icon.
2. When prompted, click "Continue" to begin the installation.
3. If you don't have Python installed, the script will direct you to download it.
   After installing Python, run the script again.
4. The script will automatically:
   - Set up the necessary environment
   - Install required dependencies
   - Launch the Portfolio Simulator application

TROUBLESHOOTING:

If you receive a warning about the script being from an unidentified developer:
1. Right-click (or Control-click) on the script
2. Choose "Open" from the context menu
3. Click "Open" in the security dialog that appears

If you need further assistance, please contact the application provider.
